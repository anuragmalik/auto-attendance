Pre-Requisites :

In order to run and use this project you need to take care of following points and contitions.
Technologies required : The development machine should contain Java JDK Version 1.7.0 or above.
Tools to be installed :
Install Eclipse IDE for development environment. Version - Indigo or above.
Install IBM Web Sphere Application Server. Version - 3 or above
Install IBM DB2 Database Community Edition. Version - 9.0 or above.
Further following plugins can be downloaded from Eclipse Market Place and used for ease of development and UML modelling -
WASCE Application Server Plugin v3.0 or above.
Object Aid UML Modelling Tool
Note : Make sure that classpath settings are done for Java Virtual Machine and WASCE Application Server as well. Further for the purpose of efficiency, developer may create a database pool for interaction between IBM DB2 & WASCE.

Please select an option from below :

Project Development Environmet - Eclipse IDE
WASCE - Application Server
Database - IBM DB2

Project Development Environment :

Steps to import the existing project into workspace of Eclipse IDE are as follows:

1. From the main menu bar, select File > Import. The Import wizard opens.
2. Collapse or click + in General > Existing Project into Workspace and click Next.
3. Choose either Select root directory or Select archive file and click the associated Browse to locate the directory or file containing the projects i.e. Select EchWebsite folder from the Source Code folder in the CD provided along with the project report.
4. Under Projects select the project or projects which you would like to import.
5. Click Finish to start the import.

Setting Application Server :

Steps for setting up a server in Eclipse IDE are as follws:

1. Right click on project folder in your Eclipse project explorer.
2.Select "Run As > Run Configurations" option in the menu.
3. On the new window, select "IBM WASCE" amongst several other options.
4. Now, select Server tab from above options and use drop down menu to select the required application server. for example we have used - "IBM WASCE v3.0 server at localhost
5. You may now simply use- "Run as > Run on Server" option to run the dynamic web project on application server.

Setting up database :

Steps to import the existing tables into an existing databse instance of IBM DB2 are as follows:

1. From the Start menu, select IBM DB2 > DB2Copy(Default)
2. Open Command Line Tools and click Command Line Processor
3. On the Command Line Window, type - connect <your database name>
4. After Successfull connection with the database, you can use the database tables provided by us to create a similar instance of your database.
For this import each of the tables provided in CD in "Source code > Database" into your databse using following command -
import from 'filename' of ixf CREATE into table_name
